EXPERIMENT NO. 9

TITLE:
Build a REST API using Express.js and MongoDB.

AIM:
To build a simple REST API using Express and perform CRUD Operations on MongoDB.

STEP 1

Open VS Code

File → Open Folder

Open Mongo DB Folder

--------------------------------

STEP 2

Initialize Project

npm init -y

--------------------------------

STEP 3

Install Dependencies

npm install express mongoose

--------------------------------

STEP 4

Start MongoDB

mongod

OR

Open MongoDB Compass

mongodb://127.0.0.1:27017

--------------------------------

STEP 5

Run Server

node app.js

Output:

MongoDB Connected

Server Running on
http://localhost:3000

--------------------------------

CREATE

http://localhost:3000/add

--------------------------------

READ ALL

http://localhost:3000/users

--------------------------------

READ SINGLE

--------------------------------
