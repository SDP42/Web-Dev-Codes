Express.js & MongoDB CRUD API

This project demonstrates Create, Read, Update and Delete operations using Express.js, MongoDB and Mongoose.

Operation URL
Create http://localhost:3000/add
Read All http://localhost:3000/users
Read Single http://localhost:3000/users/id
Update http://localhost:3000/update/id/Ram/25/ram@gmail.com
Delete http://localhost:3000/delete/id

Technologies Used

  • Node.js
  • Express.js
  • MongoDB
  • Mongoose
  • REST API
