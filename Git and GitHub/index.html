Git and GitHub Commands

This experiment demonstrates Version Control Operations using Git and GitHub.

Basic Commands

Command Description
git init Initialize Local Repository
git status Check Repository Status
git add . Add All Files to Staging Area
git commit -m "message" Create Commit
git log View Commit History
git clone URL Clone Repository
git push origin main Upload Code to GitHub
git pull origin main Download Latest Changes

Git Workflow

Working Directory
↓
Staging Area
↓
Local Repository
↓
GitHub Repository

Branch Commands

Command Description
git branch feature1 Create Branch
git checkout feature1 Switch Branch
git merge feature1 Merge Branch
git branch -d feature1 Delete Branch

Conclusion

Git is a Version Control System used to track project changes, while GitHub is an online platform used to host Git repositories and support collaboration.
